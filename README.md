## Forma de despliegue

Para desplegar páginas web estáticas en **GitHub Pages** es muy sencillo.

El primer paso consiste en crear un nuevo repositorio o, si ya tienes uno, renombrarlo.
El nombre del repositorio debe seguir el formato:

```
nombre_repo.github.io
```

Luego, debajo del nombre del repositorio, busca **Settings** y haz clic.
Se abrirá una nueva ventana.

En el menú del lado izquierdo, busca la opción **Pages** y selecciónala.
Usa la siguiente configuración:

- **Source:** Deploy from a branch

- **Branch:** main

- **Folder:** /root

Finalmente, guarda los cambios.

Después de unos minutos, tu URL estará lista y se mostrará en la misma ventana en la que te encuentras.

**Ejemplo**
URL:`https://usuario.github.io`
